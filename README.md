# PIRATE
 Python Remote Acces Tool

### Usage
1- run payloader.py to generate client script

2- run pirate.py to manage server and receive connection

### Features

- Open remote shell
- Take screenshot 
- Take webcam picture
- Run keylogger
- Windows persistence
- Send and receive files
- Open windows messagebox
